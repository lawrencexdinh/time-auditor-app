The design space

Rough directions for every MVP screen, loosely inspired by Rize but rebuilt for an ADHD uni student: one obvious next action, fewer decisions per screen, calm neutral stats (no guilt-red). ★ = the three screens we explored hardest. Use the tabs above to flip through.

① Sign up

Lowest-friction entry. Email + password, stay logged in. 2 directions.

② Tasks ★

The home base. Dense table vs. due-date groups vs. "one next thing". 3 directions.

③ Projects

Group tasks, see lifetime time spent. 2 directions. (Subjects = v2.)

④ Timer ★

The pomodoro loop. Big calm ring vs. zero-chrome vs. interval map. 3 directions.

⑤ Session setup

Pick task + goal, intervals, durations. Panel vs. wizard. 2 directions.

⑥ Check-in

Focus score 1–10 before the break. The accountability differentiator. 2 directions.

⑦ Floating bar

The movable always-on-top reminder you love. 3 directions.

⑧ Insights

Day / week / month breakdowns + average focus score. 3 directions.

before you start

Session setup & entry

Pick a task (existing or new), set a goal, choose interval length & count. If the task already has a project it autofills. After a break, the same task/goal carries into the next interval automatically.

the differentiator

Post-interval check-in

When a focus block ends, score how focused you were 1–10. It's required to start the break — one tap, friction-low. This is what makes the app an audit loop and not just another pomodoro timer.

A Tap a number 1–10

Fastest possible. Big targets, neutral framing, then "Start break".
